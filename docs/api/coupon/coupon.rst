优惠券分次核销
::::::::::::::::::::::::::::::::::::

该接口是在通过点评发放优惠券后，点评用户通过该优惠券在合作方消费后，合作方将券信息传送给点评，通知点评该优惠券被消费，以核销掉该优惠券。
