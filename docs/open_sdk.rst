.. SDK相关说明


SDK
=========================

概述
--------------------------------

点评购物开放平台SDK提供了API的请求封装、摘要签名、响应解释、消息监听等功能，使用SDK可以轻松完成API的调用，API结果的获取，消息的实时监听。

环境依赖
--------------------------------

 - JAVA SDK 需要依赖 Java SE/EE 1.6及以上 （不支持Android平台）
 - PHP SDK 需要依赖 PHP 5及以上
