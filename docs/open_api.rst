API
=========================

点评购物开放平台（DIANPING Open Platform）的API是基于HTTP协议来调用的，开发者（ISV）可以直接使用点评提供的官方SDK（支持多种语言，包含了封装的请求，签名加密，响应解释，性能优化等）来调用，也可以根据开放平台的协议来封装HTTP请求进行调用，以下主要针对自行封装HTTP请求进行API调用的原理进行详细解说。
